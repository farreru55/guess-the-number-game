# Mystic Number Guessing Game

A beautifully designed, minimalist "Guess the Number" game featuring the elegant **Catppuccin Mocha** color palette. Built with a focus on modern UI/UX principles, including glassmorphism, smooth animations, and responsive design.

## üöÄ Getting Started

Since this is a client-side web application, no installation or build process is required.

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Start guessing!

## üéÆ How to Play

1. The computer will pick a random mystery number between **1 and 10**.
2. You have **3 lives** to guess it correctly.
3. Enter your guess in the center field and press **Enter** or click **Reveal Mystery**.
4. If your guess is wrong, the input will shake and give you a hint (Higher/Lower) while you lose one heart.
5. Find the number before you run out of lives to win!
